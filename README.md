# 终末地基质计算器

基于明日方舟：终末地的基质刷取方案计算器插件。

## 功能

1. **武器名称查询** - 输入武器名称，获取推荐的刷取方案
2. **属性反查武器** - 输入属性关键词，查找匹配的武器

## 使用方法

### 武器查询
```
兔兔基质武器 典范
兔兔基质武器 遗忘
兔兔基质武器 扶摇
```

### 属性反查
```
兔兔基质 智识 攻击 夜幕
兔兔基质 力量 暴击率
兔兔基质 追袭
```

## 支持的属性关键词

### 基础属性
- 敏捷、力量、意志、智识、主能力

### 附加属性
- 攻击、生命、物理伤害、灼热伤害
- 电磁伤害、寒冷伤害、自然伤害
- 暴击率、源石技艺、终结技效率
- 法术伤害、治疗效率

### 技能属性
- 强攻、压制、追袭、粉碎、昂扬
- 巧技、残暴、附术、医疗、切骨
- 迸发、夜幕、流转、效益
